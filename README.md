# Whisper-YT

YouTube video download and transcription tool with a GUI interface.

## Features

- Download YouTube audio/video
- Transcribe audio using OpenAI Whisper
- GUI interface via tkinter
- Batch processing support

## Requirements

```
pip install -r requirements.txt
```

Also requires [ffmpeg](https://ffmpeg.org/) in your PATH.

## Usage

```
python transcribe_youtube.py
```

Or launch the GUI:
```
python gui_transcribe.py
```
